#  **G (Games Played) - 出賽數**[^1][^2]
- **定義：** 球員在比賽中上場的總場數。無論是先發、代打、代跑或防守替換，只要進入比賽場地並列入紀錄，即算一場。
- **涵義：** 衡量球員的**耐戰力 (Durability)** 與健康程度。
- **指標：** MLB 例行賽共 162 場。單季出賽 **150 場** 以上通常被視為健康的先發主力。

# **AB (At Bats) - 打數**[^1][^2]

- **定義：** 球員完成打擊的次數，但**不包含**以下情況：
    - 四壞球保送 (Base on Balls)
    - 觸身球 (Hit by Pitch)
    - 高飛犧牲打 (Sacrifice Flies) 或 犧牲觸擊 (Sacrifice Bunts)
    - 因捕手妨礙打擊而上壘
- **涵義：** 這是計算打擊率 (AVG) 與長打率 (SLG) 的**分母**。它代表球員「有機會擊出安打且未被保送」的次數。
- **注意：** 這與「打席 (Plate Appearances, PA)」不同，PA 是包含所有站上打擊區的次數。

# **HR (Home Runs) - 全壘打**[^1][^2]

- **定義：** 打者將球擊出外野全壘打牆外（界內區），或者是場內全壘打，讓打者能直接跑回本壘得分。
- **涵義：** 棒球中最具破壞力的得分手段。代表打者的**長打爆發力 (Power)**。
- **指標：** 單季 **30 轟** 為重砲手門檻；**40 轟** 為聯盟頂尖；**50+ 轟** 為歷史級別（如 Aaron Judge）。

# **RBI (Runs Batted In) - 打點**[^1][^2]

- **定義：** 打者藉由安打、全壘打、高飛犧牲打、內野滾地出局、或滿壘時的保送/觸身球，將壘上的跑者（或自己）送回本壘得分的次數。
- **涵義：** 衡量打者的**得分效率**與關鍵時刻的發揮。不過這個數據很吃隊友（隊友不上壘，就沒打點可打），因此現代數據派較不看重此數據來評估個人能力。
- **指標：** 單季 **100 分** 打點是中心打者（第3、4、5棒）的優秀標準。

# **AVG (Batting Average) - 打擊率**[^1][^2]

- **定義：** 平均每次打數能擊出安打的機率。
- 公式：
$$\text{AVG} = \frac{\text{安打數 (H)}}{\text{打數 (AB)}}$$
- **涵義：** 傳統衡量打擊能力的指標，代表**擊球準確度**。
- **指標：**
    - `.250`：聯盟平均水準。
    - `.300`：優秀打者（三成打者）。
    - `.330+`：打擊王競爭者。

# **OBP (On-Base Percentage) - 上壘率**[^1][^2]

- **定義：** 打者在一個打席中，不被解決出局（成功上壘）的機率。
- 公式：$$\text{OBP} = \frac{安打 + 保送 + 觸身球}{打數 + 保送 + 觸身球 + 高飛犧牲打}$$
- **涵義：** 現代棒球（魔球理論）最重視的數據之一。因為「不出局」是得分的前提。它比打擊率更準確，因為它**計入了保送**。
- **指標：**
    - `.320`：平均水準。
    - `.370`：優秀。
    - `.400+`：頂尖選球大師（如 Aaron Judge, Juan Soto）。

# **SLG (Slugging Percentage) - 長打率**[^1][^2]

- **定義：** 平均每一個打數能推進的壘包數。
- 公式：$$\text{SLG} = \frac{\text{壘打數 (Total Bases)}}{\text{打數 (AB)}}$$
  (註：一壘安打=1, 二壘安打=2, 三壘安打=3, 全壘打=4)
- **涵義：** 衡量打者的**破壞力**。數值越高，代表該打者越常擊出二壘打或全壘打。這不是百分比，而是平均數值。
- **指標：**
    - `.400`：平均水準。
    - `.500`：具備長打火力。
    - `.600+`：MVP 等級重砲。

# **OPS (On-Base Plus Slugging) - 整體攻擊指數**[^1][^2]

- **定義：** 上壘率與長打率的總和。
- 公式：$$\text{OPS} = \text{OBP} + \text{SLG}$$
- **涵義：** 目前公認最簡單、最能快速判斷一位打者**綜合攻擊能力**的指標。它同時考慮了「能不能上壘」和「能不能打遠」。
- **指標：**
    - `.750`：合格先發。
    - `.800`：明星球員。
    - `.900`：超級巨星。
    - `1.000+`：神獸級別。
