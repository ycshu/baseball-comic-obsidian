# 閱讀心得

## 主要心得
	個人目前是看到15回，故事內容從吾郎父親茂治的離異一直到三船棒球隊打敗了橫濱少棒隊。跟以前看漫畫不一樣的感覺是，漫畫裡有很多值得思考的細節。作者修改了一個重要的設定，就是在現實中需要花很多時間去消化的感情關係，和人類那種"有些事情說不出來"的情況，在漫畫裡都是以蠻快的速度化解。譬如，吾郎第一次在三船棒球隊和其他人打練習，一開始吾郎沒有辦法接受其他人一直失誤而去責罵對方，結果大家一開始都不想打了，後來是小森以換投手的條件挽留他們。最後也讓被換成捕手的吾郎體會到什麼叫做團隊合作，為什麼即便隊友失誤，鼓勵對方是那麼的重要。這個是現實中值得學習的一點。另一個例子，在三船隊和戶塚西隊的比賽中，宇沙美這個角色的處境和在這場比賽發生的故事，反映在現實中的家暴事件。不過，漫畫把宇沙美父親設定成心理變化可以快速轉變的角色，在比賽的最後他透過反思發現自己過去教育方式錯誤。還有一個和前面兩個例子不太一樣的概念，作者在刻畫三船隊為了重新取得禮拜天練習場的使用權，進而和業餘棒球隊比賽時，吾郎的人物設定透徹說明了"取得勝利"和"棒球熱情"的不同。吾郎是一個把"打的快樂"放在前面的人。這樣的人物設定在和橫濱少棒比賽時展現的淋淋盡致，在比賽的最後時刻，他寧可不讓川瀨涼子不要對他投保送，而是正面的對決。他不想透過這種方式取得勝利，他想要的是，不管是隊友還是對手，大家都應該全力以赴，而不是透過小手段來贏得比賽，這樣才是真正的"棒球樂趣"。
	  看了這麼多的內容，我感覺得到漫畫是現實世界的一種濃縮載體。雖然漫畫為了樂趣，添增了現實世界不太會發生的元素，但他能以圖像和簡短文字呈現出人生中的種種課題。我在國小時感覺不到這點，不過隨著人生中發生愈來愈多事情，現在回過頭看漫畫更有體會性。
	  小時候看漫畫只是看圖像的刺激和樂趣，後來漸漸改看小說。我還記得人生中看的第一套漫畫是韓國兒童文學作家金玧洙歷險記系列(法果尋寶記、美國尋寶記, etc)，當時只是看樂趣的，什麼知識科普都懶得查。而人生中看的第一本小說是"天才神祕會社"。國中時期看了美國作家丹布朗的數位密碼、理查保羅伊凡斯的魔電聯盟系列，到了高中時代看了劉慈欣的三體系列、Sally Green的禁忌之子三部曲、Jerry B. Jenkins的大滅絕三部曲，再到最近看了魯賓遜漂流記和丹布朗的大騙局。我自己感覺的小說和漫畫不同的地方是，小說利用文字描繪場景，可以描繪的簡略、描繪的細膩。譬如三體第二部黑暗森林法則的開頭利用螞蟻攀爬一座高聳石頭上的刻痕作為小說的序章。作者沒有直接點出場景，而是透過螞蟻在刻痕中爬的方向，以及遇到的障礙物，並由三體第一部的故事逐漸襯托出這是葉文潔女兒的墓碑。而漫畫則是重點就在圖像上面，且由於圖像不會動，如果是動態的畫面感就必須加上很多額外的點綴。除此之外，小說能夠把世界觀寫得很複雜，從一開始的一個點，發散到非常多件事情，再到小說的結尾收斂成一個點。但我到目前為止只認為，三體和數位密碼是把故事寫的最完整，最沒有腦補餘地的神作。這兩本著作都把故事細節寫得很細膩，每一點最後都能夠回到主線上面。其實我大部分都是看科幻系列的小說，所以作者想在現實生活中表達的都離個人生活比較遠，例如三體在講宇宙的規則和人類未來的歷史；數位密碼是講美國國家安全局的故事。至於漫畫，說真的，如果已經有動畫的話我寧可看動畫，像我從沒看過進擊的巨人漫畫，但看過他的影集。對我來說，漫畫更像是一種娛樂和反思的結合。有畫面，讀者不需要在腦中建構場景，這反倒可以讓讀者更專注在故事細節和想表達的事情上面。

## 棒球大聯盟1精彩畫面

### 1.
![[012.jpg]]
來源 : 棒球大聯盟 vol.03 012.jpg
畫面說明 : 這是本田茂治打中吉布森時速160km的直球瞬間。右邊透過許多直線條和模糊化棒球、棒子和手臂來襯托這是瞬間發生的事情。球的扭曲化程度說明了這顆球的瞬時速度超級快；左邊把鏡頭拉遠，用爆炸效果點綴擊中點，以及模糊化棒子，讓讀者自然的推測出這揮棒力道非常大。同時大大的把日文字的"匡噹!!!"貼在圖上，襯托出擊中的音效聲。
### 2.
![[068.jpg]]
來源 : 棒球大聯盟 vol.05 068.jpg
畫面說明 : 這是三船棒球隊為了爭取禮拜天練習場地使用權和業餘棒球隊比賽的結尾。即便輸了，但三船隊在場上的努力及表現打動了對方的球員和教練。最後對方願意幫忙向自治會交涉，讓這群喜歡棒球的小孩可以繼續在場上練球。畫面中注意到作者利用許多線條來說明現在雨勢很大，並利用人物旁的白色閃光強化人物所說的話和三船隊球員的驚訝。左下角那一格利用吾郎水汪的眼睛襯托出他的感動。
### 3.
![[061.jpg]]
來源 : 棒球大聯盟 vol.07 061.jpg
畫面說明 : 吾郎注意到了胖子上河內的實力，進而使三船隊在集訓期間和久喜隊展開了友誼賽。右邊作者用同時很多顆扭曲的球來描述球的快速。分割畫面左邊則是用一條吾郎丟出來的球路和貼上上河內的表情，再加上右邊的驚嘆號，來描述上河內被這樣的球速驚訝到。左邊則是小森接到球，上河內揮棒落空的畫面，並利用線條來增添震撼感和球套接到球的震動感。
### 4.
![[021.jpg]]
來源 : 棒球大聯盟 vol.08 021.jpg
畫面說明 : 集訓期間，吾郎發現了自己的精神創傷，由於吾郎投了觸身球，讓他往事浮上腦海，他爸就是被觸身球害死的。這張圖是橫濱隊教練想要激怒吾郎，讓他不要再投出因創傷而導致的變速球。利用吾郎的痛點讓他暴怒，在這頁之後，吾郎在所有人的無預警的情況下向橫濱隊教練投快速直球。右上角利用黑色陰影描繪吾郎被橫濱隊教練罵，這樣的球並非直球，如果吾郎投不出來就繼續判壞球。左邊利用特寫描繪教練和吾郎愣住的臉。值得注意的是，教練的眼睛顯示在太陽眼鏡後面，是一種對苛刻語句上的強化。
### 5.
![[016.jpg]]
來源 : 棒球大聯盟 vol.10 016.jpg
畫面說明 : 三船隊對上本牧隊的黑三角三兄弟。右邊作者利用遠距離鏡頭，並把吾郎這個人物放大在右側，同時刻劃遠距離和吾郎的表情。左上利用黑色背景加上吾郎心裡的想法襯托出吾郎的內心戲和緊張感。

### 6.
![[058.jpg]]
來源 : 棒球大聯盟 vol.11 058.jpg
畫面說明 : 三船隊和戶塚西隊的比賽來到後面，吾郎嗆宇沙美只是個父親的投球機器人。很明顯地，作者刻意把宇沙美畫成毫無情緒的樣子(左下右)。左上角吾郎也很直白的說出棒球的樂趣，可說是這一集的經典。

### 7.
![[011.jpg]]
![[14012.jpg]]
來源 : 棒球大聯盟 vol.14 011.jpg、vol.14 012.jpg
畫面說明 : 橫濱隊和三船隊比賽的最後階段。吾郎由於身體虛脫，往本壘跑的過程倒在地上，全隊在幫他加油，吾郎重新站起來，向最後的幾尺衝去。本壘手壽也接到球和吾郎採回本壘板近乎是相同時間，大家在裁判喊出結果時不知道到底是誰贏了。我想此時只有壽也、吾郎和裁判知道，這一局將成為三船隊的傳奇。
