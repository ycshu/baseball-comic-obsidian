# 閱讀心得
# 第一話 何方神聖?

對手對於本田吾郎的快速球感到震驚，看了第一球進到好球帶，揮空第二球，想著明明紅中的快速球卻揮不到球，整理好情緒將兩球快速球碰至界外，最後卻依舊被吾郎揮空三振，感受到他的厲害，但其他同隊隊友卻相當不屑，認為只是狀況不好，到了吾郎攻擊時間第一球就將對手的球擊出牆外，全壘打，整個比賽都是他自己的表演。

# 第二話 怎麼會有這種事情!?

海堂的球探到了會場，很驚訝擁有強大火力的青武館竟然落後，原本只是想觀察參加橫濱青少棒聯盟的球員，卻被本田吾郎強勁的速球所吸引，及翻閱資料卻查無此人。

# 第三話 居然會有這種人!?

吾郎對上外號清道夫的強打天野同學，上來的第一球便將其擊向界外，在緊接著一個壞球後，天野將球打向空中，吾郎接殺贏得這次對決，天野球棒也被球的後勁震斷，海堂球探大鑽賞其表現，表示一定要得到這下個世代的王牌投手，同學們開始期待可以晉級縣大賽，而下一輪的對手為棒球名校中央中學及壽也念的友浦之間的勝方。

# 第四話 新秀選拔

海堂高中的球探在吾郎回家的路上攔住了他，鄉招募他常為學校的棒球公費生，沒想到吾郎一口回絕，如果加入的話就不能和優秀的選手成為對手，並且擊敗他們，因為他不想和壽也成為隊友，則無法成為對手競技，果然在壽也優秀的表現下友浦贏下中央中學，準備對戰下一輪，原先以為青武館會出線，沒想到被吾郎無安打完封比賽，球探告訴壽也希望他放棄海堂這個選項，因為他們想得到吾郎，壽也打電話向吾郎宣戰明天輸的人要放棄海堂。

# 第五話 正式宣戰

清水勸說吾郎和壽也一起去海堂打球，因為原話被誤解，不要害壽也也無法去海棠，壽也對於是不是誤會不在乎，但對於被海棠球探表示選擇吾郎而非自己感到不滿，一心只想要擊敗他，兩人正式立下戰帖。

# 第六話 第一回合

吾郎全力投球，故意保送前一位打者要對決壽也，在專注對決的情況下跑者不斷盜壘試圖打斷節奏，但在吾郎的眼下只有壽也，最終讓其擊出高飛球被接殺，贏得第一回合的對決。

# 第七話 圈套...

對方投手看似普通但怪異的投球姿勢讓三船東吃足苦頭，犀利的變化球三振打者，但又因為壽也精采的配球騙到打者，對決上吾郎，但此時卻被壞球連發，壽也表示只想故意保送他不讓他表現。

# 第八話 保送...!?

吾郎看到壽也站起來引導投手投出大壞球，心生不滿不想要就此結束，因此積極出棒，最後擊出滾地球出局，吾郎才發現投出的是曲球，一開始就沒有要保送他只是要玩弄他的心態，讓他追打壞球，壽也跟倉本說三振後回來就好，並誓言要從吾郎手中打出全壘打得分，但上場後卻擺出短棒觸擊。

# 第九話 男子漢的對決!

回傳不及壽也打出內野安打，安全站上一壘，吾郎對於這種偷雞摸狗的上壘方式感到不屑，上壘後的壽也不斷向二壘移動，故意使吾郎分心，在受不了的時候傳向一壘牽制，卻傳了一個大爆傳，壽也站上三壘，吾郎氣到將球和球帽摔到地上，但比賽依然繼續，此時輕鬆走想本壘得分。

# 心得

本篇看到壽也和吾郎的對決，吾郎想法非常簡單就是要正面對決贏得比賽，但聰明的壽也正巧利用這個心態，屢次玩弄吾郎於手掌之中，不正面對決硬碰硬，用最輕鬆的方式贏你。
