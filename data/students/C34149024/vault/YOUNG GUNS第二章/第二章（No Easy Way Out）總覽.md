# 主軸
**早晨家庭壓力 → 進入校園的人際與秩序 → 小衝突逐步升級成更硬的對立**，整體在鋪陳「你想脫身，但環境不給你簡單退路」的基調。[[YOUNG GUNS]] [[第二章]] [[成長]]
# 開場節奏
主角一早就被時間、家人、規矩推著走，呈現「還沒出門就先被消耗」的狀態。[[家庭壓力]] [[日常崩壞]]
# 家庭
父親（或父職角色）以「教育／規矩／成材」作為權威核心，語氣強硬，家中氣氛偏緊繃；母親在其中緩衝，妹妹（莉莉）更像是家庭情緒的擺盪點。[[權威]] [[親子衝突]] [[情緒勞動]] [[莉莉]]
# 進校園後
班級與走廊裡的關係網浮現：學姊/同學/老師/小圈圈，表面是玩笑與日常，底層其實是「誰能壓誰」的排序。[[同儕]] [[校園秩序]]
# 事件推進方式
從**惡作劇、嘲諷**一路加壓到**明顯肢體衝突**，讓讀者看到暴力不是突然出現，而是被「習慣」一步步養大。[[校園霸凌]] [[暴力升級]]
