# 發想過程
## 漫畫概念
聚焦於2025 World Series中洛杉磯道奇對上多倫多藍鳥的七場比賽，摘取其中五段精采部分作為漫畫主軸，融入組員自己的創意與想法，保留賽場過程真實的同時加入背景故事與情感表達。使用小故事與球評、休息區等等作為過渡，形成一個以團隊合作和感情羈絆為核心的棒球故事。
由於賽事發生於美國（道奇球場）與加拿大（羅傑斯中心），為了還原MLB）的真實氛圍，漫畫中的對白、球評轉播、環境標語及擬聲詞皆統一使用英文呈現，藉此強化跨國對決的臨場感與美式漫畫的硬派風格。

## 開局
先用比賽的名稱表明本部漫畫的標題，接著依序介紹先發打者以及投手陣容，最後利用賽評介紹山本在上一場比賽中的精彩表現來銜接game 2的故事內容
## game2
#人物 ：YAMAMOTO (LAD) / Guerrero Jr. (TOR) / 藍鳥球迷
#時間 ：2025 WS Game 2 一局下半
#地點 ：多倫多 羅傑斯中心
#故事 ：
山本由伸踏上投手丘的那一瞬間，多倫多的主場聲浪不是「吵」，而是像一堵有重量的牆迎面壓來。藍鳥球迷的「藍色海洋」在看台上翻湧，手臂高舉、掌聲爆裂，彷彿全場都在催促他第一個失手。球還沒來得及把節奏完整裝進身體裡，危機就已經先抵達：George Springer 先敲出二壘安打，下一棒又補上一支安打，局面立刻變成無人出局、一三壘有人。這種開局最殘酷的地方在於，它不需要大錯——只要一顆球稍微偏離、稍微鬆手，整個夜晚就可能被推向失控。

那一秒，山本的腦中閃過的不是戰術，而是一種刺痛的自問：「我是不是……被看穿了？」連續兩支安打像兩道刻痕，刻在他的呼吸節奏上。他把球握進手套裡，指尖卻第一次感覺不到縫線的「熟悉」。看台的吼聲像從四面八方壓縮過來，連頭盔上方的燈光都像更白、更刺眼。捕手蹲在那裡，手套張開，像一個等他回答的問題；而他短短一瞬間，竟想不起自己平常最相信的答案。

他低下頭，帽沿壓得更低一點點，像把外界的喧鬧切掉一層。松香粉在指腹摩擦，卻仍抹不掉那個念頭：如果下一球再偏一點，這一局會不會就這樣失控？

就在自我懷疑幾乎要滲進骨頭時，一個插曲突然刺破節奏——內野看台有位山本球迷太過激動，竟把自己珍藏的「山本簽名球」用力丟進場內。球擦過來，只「敲」到山本鴨舌帽帽沿的前緣，沒有真的打到他。全場一瞬間像被嚇到般晃了一下，噪音也短暫失真。山本彎腰撿起球，看到自己的簽名，抬眼看向球飛來的方向——那位球迷雙手合十，像在道歉。山本沒有生氣，只把球交給工作人員，示意把球還回去，然後把注意力放回捕手手套那個白色的靶心。

而就在他抬頭的同時，他在那片深藍的海裡看見另一個「固定點」：有人特地從日本前來，在看台上舉起手工牌子，用日文寫著加油的字樣。筆畫不工整，卻很用力；雙手微微發抖，像怕他看不見似的，一次又一次把牌往上頂。那不是主場的聲音，也不是對手的挑釁——是一段跨過海、跨過時差，特地來到這裡的「相信」。

山本的胸口像被什麼輕輕敲了一下。不是變得熱血，而是更安靜、更深的一句提醒浮上來：我一路走到這裡，本來就不是靠世界安靜——而是靠我把雜音切掉。他吐出一口氣，讓那口氣慢到幾乎看不見。縫線回來了，觸感回來了，手指重新知道該落在哪裡。他把帽沿再壓低一點點，這一次不是逃避，而是把世界關到只剩下捕手手套那個白色靶心。

捕手比出暗號，山本點頭——他把節奏交給那顆曲球：不是靠蠻力硬碰硬，而是用時間差，把對手最想出棒的瞬間往前或往後推離。接著，Vladimir Guerrero Jr. 走進打擊區。看台像同時吸了一口氣，等著那一棒把比賽扯開。但山本用曲球把打席切碎：讓時間感失真、讓揮棒提早一步落空。當判決落下——三振。

更關鍵的是，山本沒有停下。他把剛才那一瞬間找回來的「自己」，原封不動地複製到下一位打者身上：同樣的呼吸、同樣的帽沿陰影、同樣的指尖觸縫，像把節奏鎖死。再送出一次三振。兩個連續三振像兩道門閂，把原本可能爆開的主場氣勢硬生生扣回原地；而在觀眾席另一側，零星的道奇支持者開始站起來、握拳、喊聲穿過噪音——因為所有人都看得出來：山本不是在撐過危機，他正在用自己的方式，把這場比賽的前奏改寫成他的節拍。
## game3-1
#人物 ：KERSHAW (LAD) / Max Scherzer (TOR 教練) / 大谷翔平 (LAD)
#時間 ：2025 WS Game 3 十二局上半
#地點 ：洛杉磯 道奇球場
#故事 ：
Kershaw 曾以為這場「退役戰」會像教科書般優雅地落幕。 但命運在第十二局開了一個殘酷的玩笑。滿壘，兩出局，分差僅在一線之間。當道奇隊的牛棚徹底燃盡，原本坐在板凳上準備迎接告別儀式的 Kershaw，緩緩站了起來。

這不只是為了一場勝負。他隔著球場，望向藍鳥隊休息區。那裡站著 Max Scherzer——那個和他一起在德州泥土地上長大，卻在高中畢業後因理念不合而分道揚鑣的男人。幾十年來，他們在球場上像火藥般一碰就炸，冷戰與對抗成了他們唯一的對話方式。

如今，Max Scherzer 已披上教練服，而這是 Kershaw 職業生涯最後的五分鐘。

「老兵不該這樣死在戰壕裡，」教練 Roberts 眼神複雜，但 Kershaw 已經開始拆開肩膀上的冰袋。「這是我選的死法，Dave。」

他忍著經年累月的舊傷劇痛走上投手丘。每跨一步，左肩都像有鋼釘刺入。Max Scherzer 走出休息區，站在場邊死死盯著投手丘。兩人的視線在空中交火，那是跨越了二十年、從兒時友誼到職業宿怨的重量。

「來吧，Clayton。」Max Scherzer 無聲地動了動嘴唇，「讓我看看你的執著到底值多少。」
Kershaw 拿出了那標誌性的高抬腿動作，像是一座古老而莊嚴的鐘擺。球脫手而出的瞬間，帶著他對棒球的所有憤怒、愛與遺憾。那顆球在滿壘的巨大壓力下，精準地沒入了捕手手套最深處。

三振。三出局。

整座道奇球場的震動幾乎要撕裂夜空。Kershaw 沒有慶祝，他只是站在丘上，劇烈地喘息著，左手無力地垂下，像是燃盡了最後一絲生命。
他在漫天彩帶中走向球員通道，Max Scherzer 就在那裡等著他。沒有媒體預想的火藥味，兩位白髮漸生的老對手並肩站著。

「那一球，投得跟高中時一樣爛。」Max Scherzer 低聲說，眼神卻是前所未有的柔和。 「但你到現在還是打不到，對吧？」Kershaw 露出了一抹疲憊卻解脫的微笑。

大谷翔平在一旁靜靜地看著這一幕，他意識到，這不僅僅是一場救援成功，這是一個時代的兩極，在最後一刻終於回到了起點。

「那是我的青春，」Kershaw 回過頭對大谷說，「一個這輩子都不想再遇到，卻又最想再贏一次的人。」

那天，燈光熄滅後，兩人的影子交疊在通道盡頭，就像二十年前那個德州的午後一樣。
## game3-2
#人物 ：FREEMAN (LAD) / LITTLE (TOR) / 道奇隊友
#時間 ：2025 WS Game 3 十八局下半 LAD V.s. TOR 5-5
#地點 ：洛杉磯 道奇球場
#故事 ：
凌晨兩點的道奇球場，空氣濕冷得像能擰出水。 記分板上的「18」與兩顆代表出局的紅燈，在黑夜中閃爍著孤獨的光暈。這場比賽已經超越了體力的極限，變成了一場永無止盡的折磨。

藍鳥隊投手 Little 彎著腰，汗水如雨下，他只想投完這最後一個球，結束這場噩夢。而在他對面，Freeman 正緩緩踏入打擊區。與以往不同，他的腳步異常沉穩——沒有傷痛，沒有雜念。

「結束它吧。」Freeman 心裡默唸。

Little 開始抬腿。那一瞬間，空氣中出現了奇異的既視感。那一球的軌跡，竟與 2024 年那記傳說中的全壘打重疊。Freeman 的大腦在那百分之一秒內震碎了所有回憶的雜訊。

根基穩固。鎖定。釋放！

木棒擊中皮球的聲音不再是碰撞，而是一次靈魂的炸裂。 「CRACK!!!!」

Little 在球飛出的那一刻便崩潰了，他頹然地蹲在投手丘上，雙手抱頭，臉深深埋進手套。他甚至不用回頭，就知道那是屬於他的終結。「It's gone.」他絕望地心想。

球像一顆白色流星，在全場幾萬名觀眾仰視的目光中，劃破了洛杉磯漆黑的夜空。它飛越了棕櫚樹，飛越了巨大的燈柱，直到一名穿著灰色連帽衫的球迷在全壘打牆外穩穩將它接住。

「HE DID IT AGAIN!!!!!!」廣播員的嘶吼聲幾乎要撕裂麥克風。

Freeman 在紅土間奔跑，雙臂完全張開，手中的頭盔指向天空。他臉上洋溢著純粹的狂喜。而休息區早已失控，大谷翔平仰頭全力吶喊，Mookie 跳過護欄，Kershaw 緊握拳頭咆哮，這群男人像孩子一樣衝向本壘，準備迎接他們的英雄。

道奇球場陷入了藍色的瘋狂海洋，無數紙花在強光中起舞。這場長達十八局的長夜，終於在黎明前夕，迎來了最璀璨的終點。
## game6
#人物 ：KIKÈ HERNANDEZ (LAD) / buddy (the dog)/ TOR (對手)/Dodgers 同隊隊友
#時間 ：2025 WS Game 6 九局下半 LAD V.s. TOR 3-2
#地點 ：多倫多 羅傑斯中心
#故事 ：
Kiké 的手臂幾乎沒了知覺。

在外野的每一次熱身傳球，都讓他覺得手臂隨時會徹底報廢，但他依然堅守在左外野。「我會完成我的任務。不惜一切代價。」
第六戰。系列賽以 3 比 2 落後。輸了，就全都結束了。

打者站進了打擊區。Kiké 緩緩向後退了幾步。他渴望這一切是有意義的。

當球投出的那一刻，他想起了 Buddy。以前，Buddy 總會咬著球在門口等他，即便那時牠的雙腿已漸漸不聽使喚。牠從不懂什麼叫提早放棄，只要還能動，就要繼續玩下去。
擊球的聲音聽起來太尖銳、太響亮。球呼嘯著飛向左外野。Kiké 拔腿狂奔，每跨出一步，痛楚便隨之炸裂，還沒跑到警戒區，他的手臂早已灼燒般劇痛，全壘打牆飛快逼近，他縱身一躍，將身體延展到了極限。

手套確實地攔截了那一球。

他感到肺裡的空氣瞬間被撞空，身後某處，一名跑者正試圖衝壘。Kiké 撐起身體——沒有力氣，沒有知覺，全憑本能傳出。
出局。 比賽結束。

Kiké 在草地上彎下腰，那隻手臂無力地垂在身側，任由全場的歡呼聲如巨浪般將他淹沒。隊友們湧出休息區。有人抓著他的球衣，有人在他耳邊嘶吼。Kiké 只是露出了一抹微小而疲憊的笑容。
因為那一傳，他們贏得了明天。

那天夜裡，他做了一個夢。那是一座寬廣、明亮的公園，陽光灑滿每個角落，Buddy 站在草地上，比 Kiké 記憶中的樣子還要年輕，尾巴興奮地擺動著，就像以前遊戲即將開始時那樣。他們一起玩耍，直到他意識到這是一場夢。接著，Buddy 停了下來，直視著他的雙眼。沒有言語——只有那個眼神，那個總是代表著同一種含義的眼神。
做得好。

Kiké 猛抽一口氣醒了過來，房間裡一片寂靜。Buddy 早已不在了，那條項圈靜靜地躺在床頭櫃上，晨光透過窗戶悄悄爬進屋內，柔和而金黃。

另一場比賽，即將開始。
## game7
#人物 ：YAMAMOTO (LAD) / KIRK (TOR) / 現場觀眾
#時間 ：2025 WS Game 7 十一局下半 LAD V.s. TOR 5-4
#地點 ：多倫多 羅傑斯中心
#故事 ：
Yamamoto 的指尖傳來陣陣冰冷的刺痛。 那是體力耗盡與神經緊繃交織的訊號。第十一局，兩出局滿壘。在世界大賽第七戰的這一刻，投手丘成了全世界最孤獨、也最灼熱的荒島。他深吸一口氣，強烈的照明燈光將內野照得慘白，他抬起頭，視線穿過紛亂的塵埃，鎖定了打擊區上的那個男人——Kirk。

那是他自高中起就無法割捨的名字。

當球扣在掌心的那一刻，Yamamoto 想起了那些年紅土球場上的風。想起雨天一起收器材時，兩人指尖無意間的觸碰；想起那些只敢在手機螢幕上留下的「下次再戰」。他所有的壓抑、所有的暗戀、以及那些在深夜裡反覆咀嚼的思念，在這一刻全部濃縮進這顆五英兩重的球裡。

「這球，是為你投的。」他心裡默唸。

投手丘下，Kirk 緊握著球棒。他的目光從未離開過投手丘，甚至不在乎球落下的位置。他在意的只有 Yamamoto 的注視。他愛著這個冷靜得近乎殘酷的投手，愛到即便這是一個毀滅球隊夢想的陷阱，他也會毫不猶豫地揮棒。

球脫手而出。

那一球帶著決絕的尾勁，彷彿劃破了時間。Kirk 跨步、轉身，全身的力氣灌注在木棒上。然而，在球與木頭接觸的瞬間，一聲清脆得讓人心碎的斷裂聲蓋過了全場的呼吸——球棒碎了。
斷裂的木屑在空中飛舞，球軟弱無力地滾向內野。

雙殺。比賽結束。

道奇隊瘋狂地湧入球場中心，慶祝這得來不易的世界冠軍。Yamamoto 站在投手丘上，沒有歡呼，只是靜靜地看著那個站在打擊區、手中僅剩半截殘餘木棒的身影。Kirk 的眼神依然鎖定著他，那種眼神跨越了勝負，跨越了奪冠者的狂喜與失敗者的落寞。

那一刻，他們在萬人沸騰的寂靜中，完成了一場只有彼此才懂的告白。

賽後，狹窄且光影昏暗的球員通道裡，人聲漸遠。 Yamamoto 靠在牆邊，看著 Kirk 緩緩走來。兩人的球衣都沾滿了塵土，那是十一局鏖戰後的殘骸。他們短暫地對視，沒有擁抱，也沒有眼淚，所有的情感都藏在疲憊的眼底。

「下次再戰。」Kirk 聲音沙啞，那是對失敗的釋懷，也是對未來的渴求。 「一定。」Yamamoto 輕聲回應，那是承諾，也是兩人之間永不謝幕的序曲。

長廊外的歡慶聲如巨浪般模糊，但這一刻的安靜，專屬於他們。愛就像棒球，不在於終點，而在於下一次相遇時，依然願意為對方使出全力。
## 結尾
當最後一個出局數塵埃落定，道奇隊全體球員瘋狂地衝向投手丘，緊緊擁抱在一起。在這片藍色的歡騰中，每個人都看見了彼此眼中的淚水與汗水——山本與 Kirk 隔空交會的深情、Kershaw 卸下重擔的釋懷、以及 Kiké 與 Freeman 超越傷痛的堅持。他們緩緩走上頒獎台，在漫天飛舞的紙花中共同高舉冠軍獎盃，那一刻，所有的個人榮辱都融化成了團隊的榮耀。
「我們總以為棒球是數據與紀錄的堆疊，但今晚我們看見了它更真實的模樣。這是一場由無數情緒、自我成長與深刻羈絆交織而成的長征。棒球從來不是一個人的英雄主義，而是無數人為了同一個夢想，在痛苦中互相扶持、在絕境中彼此相信。所有的淚水與堅持，最終都在這座頒獎台上匯聚成了完美的終章——這就是團隊運動的真諦。」
